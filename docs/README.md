# Dokumentasi

Ringkasan dokumen penting untuk project ini:

- `docs/CONFIG.md`: konfigurasi `.env` dan variabel lingkungan.
- `docs/API.md`: daftar endpoint dan contoh request.
- `docs/DASHBOARD.md`: cara akses dashboard dan query params.

## Mulai Cepat

```bash
cp .env.example .env

deno task dev
```

## Kualitas Kode

```bash
deno task lint

deno task format

deno task test
```
